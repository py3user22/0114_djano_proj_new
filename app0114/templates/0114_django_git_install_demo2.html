1228 to add remote repositories from your git bash cli
----------------------
1- go to your github webpage & create a new repository
2- create a copy of the path url to the new repository
3- optional: if havent already create a new token in github for access or regenerate one

4- in git bash, cd to directory want to connect to the remote repo on github

5- if havent already to initialize the directory for version control
--->> git init

6- add a remote location for your github repo
_____#note: add the .git at end of path link
>> git remote add origin < repo url path from github>.git


7- check current git branch # must match the branch name on github webpage
_____# example is master for local, main for remote
---> git branch

8- verify using cmd to show the fetch and push URLs for each remote
---> git remote -v

9- to update the remote URL of repository
---> git remote set-url origin https://new.url/repo.git

10- if need to rename the current branch to match the remote repo
_____note: default to local is master and default to remote is main
>> git branch -m main
>> git push -u origin main

_____ to change the remote branch to match the local
>> git remote set-url origin https://github.com/py3user22/1228_django_renderers_demo.git master
>> git push -u origin master
